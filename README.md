## Illuminate based PHP MVC framework
- Simple todo list app

### Requirements
 - PHP 8.^

### Installation
- Run migration.sql file
```
cp .env.example .env
```

- Go to root dir
```
php -S localhost:8000 -t public
```
- Run on http://localhost:8000
